## Canvaとは？

Canvaは、**デザイン初心者向けのオンラインデザインツール**です。

テンプレート数が豊富で、ドラッグ&ドロップで誰でも簡単にプロフェッショナルなデザインが作成できます。

さらに最近では、**AIで画像を自動生成し、そのままデザインに組み込める機能**も追加されました。

| 料金 | 無料版あり / Canva Pro $120/年 |

## Canvaを始める3ステップ

### ステップ1：アカウント登録

1. [Canva公式サイト](https://www.canva.com)にアクセス
2. 「登録」をクリック
3. メールアドレスまたはGoogleアカウントで登録
4. メール確認で完成

### ステップ2：用途を選択

ログイン後、画面上部に「テンプレートを検索」というボックスがあります。

```
SNS投稿、チラシ、ポスター、プレゼン資料、
ブログバナー、YouTubeサムネイル
```

など、作成したい用途を選択します。

### ステップ3：テンプレートを選択

選択した用途に応じて、**複数のテンプレートが表示**されます。

気に入ったものをクリックすると、エディター画面に進みます。

## Canva AIで画像を生成する方法

### 方法1：テンプレートからAI画像生成

1. テンプレートを選択後、エディター画面を開く
2. 左側メニューから「Magic Design」をクリック
3. 説明文を入力（日本語OK）
4. スタイルを選択（フォト、イラスト、アニメなど）
5. 「生成」ボタンをクリック

**例：**
```
青い空とヒマワリ畑。
太陽が昇っている。
温かみのある色合い。
```

30秒程度でAIが自動でデザイン案を複数生成してくれます。

### 方法2：既存デザインに画像を追加

1. テンプレートの画像部分を削除
2. 左側メニューの「素材」 > 「AI画像」をクリック
3. 説明文を入力
4. 「生成」

すると、入力した説明に基づいて、画像がその枠にちょうど合うサイズで自動生成されます。

## Canvaで高品質なデザインを作るコツ

### コツ1：テンプレートの基本構成は変えない

テンプレートは、プロのデザイナーが作った配置になっています。

**初心者のコツは「テンプレートの枠組みは変えず、テキストと画像だけ変える」** です。

### コツ2：フォント数は最大3種類まで

複数のフォントを使いすぎると、素人っぽくなります。

```
タイトル：1種類の大きなフォント
本文：1種類の小さなフォント
強調：1種類の異なるフォント
```

最大3種類でまとめると、プロっぽい仕上がりになります。

### コツ3：色合いは統一する

テンプレートのカラーパレットを活かし、追加する要素も同じ色系を使うと統一感が出ます。

Canvaには「色を選択」機能があり、テンプレートから色を抽出できます。

<div class="success-box">
<p><strong>おすすめ</strong></p>
<p>配置とフォント、色の3つだけ意識して、それ以外はテンプレートに任せると、初心者でも素人っぽくない仕上がりになります。</p>
</div>

### コツ4：AI画像は「細部まで」と指定する

AI画像生成時に「細部まで描写」「高品質」などを追加すると、より精度が上がります。

**例：**
```
カフェの店内。
朝日が窓から差し込んでいる。
コーヒーカップ。細部まで描写。高品質。
```

## よくある質問

**Q. Canva無料版でAI画像生成はできますか？**

A. 限定的にできます。ただし、Pro版ではより多くのAI機能が使える傾向があります。

**Q. 生成した画像をダウンロードして、別のツールで使用できますか？**

A. はい、可能です。PNGやJPEG形式でダウンロードでき、どのツールでも使用可能です。

**Q. モバイルアプリでもAI機能は使えますか？**

A. はい、Canvaモバイルアプリでも同じAI機能が利用可能です。

**Q. テンプレートを複数組み合わせることはできますか？**

A. はい、別々のテンプレートで作成したデザインを、コピー&ペーストで組み合わせることも可能です。
