AIに質問しても、期待通りの答えが返ってこない...。

それ、実は**プロンプト（AIへの指示文）の書き方**が原因かもしれません。

この記事では、AIの回答精度を劇的に上げる5つのコツを初心者向けに解説します。

![AIにプロンプトを入力している様子](/ainavi-jp/images/ai-prompt-input.jpg)

## プロンプトとは？

プロンプトとは、**AIに対する指示や質問**のことです。

例えば、「東京の観光スポットを教えて」「この英文を訳して」などがプロンプトに当たります。

プロンプトの書き方次第で、AIの回答の質が大きく変わります。

## コツ1：具体的に指示する

AIは「わかりやすく」「簡潔に」などの曖昧な指示だと、何を求めているか判断できません。

**具体的な条件を明示する**ことで、精度が上がります。

### 悪い例と良い例

| 悪い例 | 良い例 |
|---|---|
| 「わかりやすく説明して」 | 「小学生でもわかるように、専門用語を使わずに説明して」 |
| 「短くまとめて」 | 「3行以内でまとめて」 |
| 「東京の観光スポットを教えて」 | 「東京の観光スポットを5つ、初めて東京に来る人向けに紹介して」 |

このように、**対象者、文字数、個数**など、具体的な条件を追加すると回答の質が向上します。

<div class="info-box">
<p><strong>ポイント</strong></p>
<p>「誰に向けて」「どのくらいの長さで」「いくつ」など、数値や条件を明示すると、AIは迷わず回答できます。</p>
</div>

## コツ2：役割を与える

AIに「あなたは〇〇です」と役割を与えると、その専門性に応じた回答をしてくれます。

これは**ロールプレイ型プロンプト**と呼ばれるテクニックです。

### 具体例

```
あなたはプロの英語教師です。
以下の英文の文法ミスを指摘し、正しい英文に直してください。

I go to school yesterday and meet my friend.
```

**AIの回答：**

```
【文法ミス】
1. "go" → "went"（過去の出来事なので過去形）
2. "meet" → "met"（過去形）

【正しい英文】
I went to school yesterday and met my friend.
```

このように、**役割を与えることで専門的な視点での回答が得られます**。

他にも以下のような役割が有効です。

- 「あなたはプロのコピーライターです」（広告文作成）
- 「あなたは優秀なプログラマーです」（コード生成）
- 「あなたは親切なカスタマーサポート担当です」（問い合わせ対応文）

![役割を与えて専門的な回答を得る](/ainavi-jp/images/ai-prompt-role.jpg)

## コツ3：出力形式を指定する

AIの回答を「箇条書き」「表形式」「ステップバイステップ」など、**形式を指定する**と読みやすくなります。

### 具体例

```
ChatGPTとClaudeの違いを、表形式で比較してください。

【項目】
- 料金
- 特徴
- 得意なこと
```

**AIの出力：**

| 項目 | ChatGPT | Claude |
|---|---|---|
| 料金 | 無料版あり / 有料版$20/月 | 無料版あり / 有料版$20/月 |
| 特徴 | 会話のテンポが良い | 丁寧で詳しい説明 |
| 得意なこと | 短文生成、アイデア出し | 長文の要約、文章添削 |

このように、形式を指定するだけで**視覚的にわかりやすい回答**が得られます。

<div class="success-box">
<p><strong>おすすめ</strong></p>
<p>比較したいときは「表形式で」、手順を知りたいときは「番号付きリストで」、要点を知りたいときは「箇条書きで」と指定しましょう。</p>
</div>

## コツ4：例を示す（Few-shot prompting）

AIに「こういう形式で答えてほしい」という例を先に示す方法です。

これを**Few-shot prompting（少数例プロンプト）**と言います。

### 具体例

```
以下のような形式で、商品レビューを書いてください。

【例】
商品名：ワイヤレスイヤホン
評価：★★★★☆
良い点：音質が良い、バッテリー長持ち
悪い点：ケースが大きい

では、以下の商品についてレビューを書いてください。
商品名：スマートウォッチ
```

**AIの出力：**

```
商品名：スマートウォッチ
評価：★★★★★
良い点：通知が便利、健康管理機能が充実
悪い点：バッテリーが1日しか持たない
```

このように、**例を示すことで、求める形式通りの回答が得られます**。

## コツ5：段階的に質問する

複雑な質問を一度に投げるのではなく、**段階的に質問を重ねる**ことで精度が上がります。

### 具体例

**悪い例：**
```
ChatGPTを使ってExcelの作業を効率化する方法を教えて。
```

**良い例：**
```
1. まず質問: ChatGPTでExcelの作業を効率化できますか？
2. AIの回答: はい、できます。関数の提案、データ整理、レポート作成などが可能です。
3. 次の質問: では、Excelの関数を提案してもらう方法を具体的に教えてください。
```

このように、**一度に全部聞かず、会話を重ねる**ことで、より具体的な回答が得られます。

![段階的に質問を重ねる様子](/ainavi-jp/images/ai-prompt-step.jpg)

<div class="info-box">
<p><strong>ポイント</strong></p>
<p>AIとの会話は一問一答で終わらせず、追加質問を重ねることで理解が深まります。「もっと詳しく」「具体例を教えて」などの追加質問も効果的です。</p>
</div>

## 良いプロンプトは再利用できる

一度作った良いプロンプトは、**テンプレートとして保存しておく**と便利です。

例えば、議事録作成、メール作成、翻訳など、よく使う作業のプロンプトは保存しておきましょう。

### プロンプトテンプレート例

**議事録作成用：**
```
以下のメモを元に、議事録を作成してください。

【会議情報】
- 日時：
- 参加者：
- 議題：

【メモ】
（ここにメモを貼り付け）

【形式】
- 決定事項、TODO、次回会議日程を明確に分けて記載
- 箇条書きで読みやすく
```

このテンプレートに毎回情報を入れるだけで、同じ品質の議事録が作れます。

## ChatGPTとClaude、どちらを使うべきか

プロンプトの使い方は、ChatGPTでもClaudeでもほぼ同じです。

| AI | 特徴 |
|---|---|
| **ChatGPT** | 短い質問に素早く回答。テンポよく会話できる |
| **Claude** | 長文の質問や複雑な指示に強い。丁寧な回答 |

個人的には、短い質問は[ChatGPT](/ainavi-jp/posts/chatgpt-beginner-guide/)、長文や複雑な依頼は[Claude](/ainavi-jp/posts/claude-beginner-guide/)がおすすめです。

## よくある質問

**Q. プロンプトは日本語でも英語でも精度は変わりませんか？**

A. 2026年2月時点では、日本語でも英語でもほぼ同じ精度です。ただし、専門的な内容は英語の方が精度が高い場合があります。

**Q. プロンプトが長すぎると、AIは混乱しませんか？**

A. いいえ、むしろ詳細に指示した方が精度が上がります。ただし、指示が矛盾していると混乱するので、条件は明確にしてください。

**Q. AIが求める回答をしてくれない場合はどうすればいいですか？**

A. 「もっと具体的に」「別の言い方で」「例を3つ挙げて」など、追加で指示を出してみてください。何度か質問を重ねると精度が上がります。

**Q. プロンプトのコツを学べる本やサイトはありますか？**

A. 2026年2月時点では、プロンプトエンジニアリングに関する書籍や無料の学習サイトが増えています。「プロンプトエンジニアリング 入門」で検索してみてください。

## まとめ

この記事で紹介した内容をまとめます。

- プロンプトは具体的に書く（対象者、文字数、個数を明示）
- AIに役割を与えると専門的な回答が得られる
- 出力形式を指定すると読みやすくなる
- 例を示す（Few-shot prompting）と求める形式通りになる
- 段階的に質問を重ねると精度が上がる
- 良いプロンプトはテンプレート化して再利用する

まずは[ChatGPT](/ainavi-jp/posts/chatgpt-beginner-guide/)で、この5つのコツを試してみてください。

プロンプトの書き方を変えるだけで、AIの回答の質が劇的に変わることを実感できるはずです。

他にも、[Excelの作業](/ainavi-jp/posts/ai-excel-howto/)や[メール作成](/ainavi-jp/posts/ai-email-writing/)でAIを活用する際にも、このプロンプトのコツが役立ちます。
