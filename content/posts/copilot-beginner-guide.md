「AIに興味はあるけど、新しいサービスに登録するのは面倒」という方へ。
Windows PCを使っているなら、**Microsoft Copilot（コパイロット）**がすでに使える状態になっています。

## Copilotとは？

Copilotは、**MicrosoftがWindowsに標準搭載したAIアシスタント**です。
ChatGPTのベースとなっている技術（GPT-4）を使っています。

| 項目 | 内容 |
|------|------|
| 開発元 | Microsoft（米国） |
| 料金 | 無料 / 有料は月額3,200円（Copilot Pro） |
| 使い方 | Windows標準搭載 / ブラウザ（copilot.microsoft.com） |
| 日本語 | 対応済み |

**最大の特徴は「インストール不要」**です。
Windows 11なら、タスクバーのCopilotアイコンをクリックするだけで起動します。

## Copilotを使う方法

### Windows PCの場合

1. タスクバーのCopilotアイコンをクリック（またはキーボードのWindowsキー + C）
2. チャット画面が表示されるので、質問を入力して送信する

アカウント登録もメール認証も不要です。

### ブラウザの場合

1. **copilot.microsoft.com** にアクセス
2. Microsoftアカウントでログイン（なくても一部機能は使える）
3. チャット画面から質問する

## 無料で何ができる？

### できること

- **文章の作成** — メール、報告書、SNS投稿文
- **質問への回答** — Bing検索と連動した回答（出典付き）
- **画像の生成** — DALL-E 3を使った画像作成
- **Office連携**（有料） — Word・Excel・PowerPointでAIを使える

### 他のAIと比べた強み・弱み

| 項目 | Copilotの評価 |
|------|-------------|
| 始めやすさ | **最も手軽**（Windows標準・登録不要） |
| 回答の出典表示 | **リンク付きで表示される** |
| 日本語の品質 | ChatGPTとほぼ同等 |
| 長文処理 | [Claude](/ainavi-jp/posts/claude-beginner-guide/)に劣る |
| 画像生成 | [ChatGPT](/ainavi-jp/posts/chatgpt-beginner-guide/)と同等（DALL-E 3） |

**Copilotの一番の強みは「始めるハードルの低さ」**です。
アカウント作成なしで、今すぐAIを試せます。

## 初心者におすすめの使い方3つ

**1. ネット検索の代わりに使う**

「来週の東京の天気は？」「iPhone 16の発売日はいつ？」など、調べものに使えます。
回答にBing検索の出典リンクが付くので、情報の確認もしやすいです。

**2. 文章を書いてもらう**

「退職届の書き方を教えて」「忘年会の案内メールを書いて」のように指示すると、すぐに使える文章が出てきます。

**3. 画像を作ってもらう**

「夕日に照らされた富士山の水彩画風イラスト」のように指示すると、画像が生成されます。
無料で1日15回程度の画像生成ができます。

## まとめ

- CopilotはWindows標準搭載のAI。登録不要ですぐ使える
- 回答に出典リンクが付くので、情報の確認がしやすい
- 画像生成にも無料で対応
- Windows PCを使っているなら、タスクバーのCopilotアイコンをクリックして試してみてください
