プレゼン資料を作るのに時間がかかる。何を書けばいいかわからない。そんな悩みはAIが解決してくれます。

この記事では、ChatGPTやCopilotを使ってプレゼン資料を作る方法を、初心者向けに解説します。

## AIでプレゼン資料のどこまで作れる？

AIは以下のようなプレゼン資料作成を手伝ってくれます。

- **構成案の作成**（何をどの順番で話すか）
- **スライドタイトルの生成**
- **箇条書きポイントの作成**
- **話す内容のスクリプト生成**
- **グラフ・図表の説明文**

完全自動ではデザインはできませんが、「何を書くか」の部分は大幅に時短できます。

## ChatGPTでプレゼン構成を作る方法

ChatGPTを使ったプレゼン資料作成の手順を見ていきましょう。

### 1. テーマと目的を伝える

まずはプレゼンのテーマと目的を伝えます。

**例文:**

```
「新商品のマーケティング戦略」について、
社内向けに10分間のプレゼンをします。
スライド構成を考えてください。
```

**ChatGPTの回答:**

```
以下のような構成をおすすめします：

1. 表紙（商品名・発表者）
2. 背景・課題（なぜこの商品が必要か）
3. 商品概要（特徴・強み）
4. ターゲット顧客（誰に売るか）
5. マーケティング戦略（どう売るか）
6. 予算・スケジュール
7. 期待される効果
8. まとめ・次のアクション

各スライドの詳細も作成しましょうか？
```

このように、全体の流れを提案してくれます。

### 2. 各スライドの内容を生成する

構成が決まったら、各スライドの内容を作ってもらいます。

**例文:**

```
3枚目「商品概要」のスライドの内容を、
箇条書き3つで作ってください。
```

**ChatGPTの回答:**

```
【商品概要】

- 軽量かつ高性能なワイヤレスイヤホン
- 最大36時間の連続再生が可能
- ノイズキャンセリング機能で集中できる環境を提供

補足説明:
この商品は、移動中でも快適に音楽や通話を楽しめる設計です。
```

箇条書きだけでなく、話す内容の補足も作ってくれます。

### 3. スピーカーノートを作る

プレゼンで話す内容（スピーカーノート）も生成できます。

**例文:**

```
上記のスライドで話す内容を、
1分間のスクリプトにしてください。
```

**ChatGPTの回答:**

```
「それでは、商品概要についてご説明します。

今回ご紹介するのは、軽量かつ高性能なワイヤレスイヤホンです。
重さはわずか4グラムで、長時間つけていても疲れません。

また、最大36時間の連続再生が可能なため、
1週間充電なしで使い続けることができます。

さらに、ノイズキャンセリング機能により、
周囲の雑音をカットし、集中できる環境を提供します。」
```

このスクリプトを参考に、自分の言葉で話せばOKです。

ChatGPTの詳しい使い方は[ChatGPTの使い方｜登録から初めての質問まで3分で解説](https://ainavi416.github.io/ainavi-jp/posts/chatgpt-beginner-guide/)をご覧ください。

## Microsoft CopilotでPowerPointを操作する

Microsoft CopilotはPowerPointと連携できるので、より効率的にプレゼン資料が作れます。

### 1. Copilotを起動する

Windows 11のタスクバーからCopilotを起動します。

### 2. プレゼン資料の構成を聞く

**例文:**

```
「AIの業務活用事例」について、
営業向けに15分のプレゼンをします。
PowerPointのスライド構成を教えてください。
```

**Copilotの回答:**

```
以下の構成をおすすめします：

1. タイトルスライド
2. 目次
3. AIとは？（基礎知識）
4. 営業でのAI活用例①（メール自動生成）
5. 営業でのAI活用例②（顧客分析）
6. 営業でのAI活用例③（提案資料作成）
7. 導入のメリット
8. 導入の流れ
9. まとめ・質疑応答

PowerPointで新しいプレゼンテーションを開き、
このスライド構成で進めてください。
```

### 3. 表やグラフの説明文を作る

データを見せる際の説明文も作ってもらえます。

**例文:**

```
「2024年と2025年の売上比較」を示すグラフがあります。
2024年は500万円、2025年は800万円でした。
このグラフの説明文を作ってください。
```

**Copilotの回答:**

```
【売上推移】

2024年から2025年にかけて、売上は500万円から800万円に増加しました。
これは前年比60%の成長を示しており、
新規顧客の獲得とリピート率の向上が主な要因です。
```

Copilotの詳しい使い方は[Microsoft Copilotの使い方｜Windows標準のAIを活用する方法](https://ainavi416.github.io/ainavi-jp/posts/copilot-beginner-guide/)をご覧ください。

## プレゼン資料作成のコツ

AIにプレゼン資料を作ってもらうときのコツを紹介します。

### 1. 聞き手と時間を明確にする

「誰に向けて、何分間話すか」を伝えると、適切な構成を提案してくれます。

**良い例:**

```
「中小企業の経営者向けに、5分間でAI導入のメリットを伝えたい」
```

**悪い例:**

```
「AIについてプレゼンします」
```

聞き手と時間を伝えることで、詳しさや構成が最適化されます。

### 2. 1枚のスライドに1つのメッセージ

AIに依頼するときも、「1枚のスライドに1つのメッセージ」を意識しましょう。

**例:**

```
「コスト削減」のスライドを1枚作ってください。
箇条書き3つで、簡潔にまとめてください。
```

情報を詰め込みすぎず、シンプルに作ることが大切です。

### 3. AIが作った内容を自分の言葉に変える

AIが生成した文章をそのまま使うのではなく、自分の言葉に直しましょう。

**AIの文章:**
> 「本製品は、高度な技術により実現された革新的なソリューションです。」

**自分の言葉:**
> 「この商品は、最新技術を使った新しい解決策です。」

自分らしい表現に変えることで、説得力が増します。

## よくある質問

### Q1: AIでPowerPointのデザインまで作れますか？

A: 現時点では、AIは文章の生成が中心で、デザインは自分で作る必要があります。ただし、Canva AIなどのツールを使えば、AIでデザインテンプレートを選ぶことも可能です。

### Q2: プレゼンのリハーサルもAIでできますか？

A: ChatGPTに「この内容で質問されそうなことを教えて」と聞けば、想定質問を作ってくれます。リハーサルの準備に役立ちます。

### Q3: 英語のプレゼン資料も作れますか？

A: はい、「英語でプレゼン構成を作ってください」と依頼すれば、英語の資料も生成できます。翻訳も同時に依頼できます。

## まとめ

AIを使えば、プレゼン資料作成の時間を大幅に短縮できます。

**この記事のポイント:**

- AIで構成案・スライド内容・スピーカーノートを自動生成できる
- 聞き手と時間を伝えると、適切な構成が提案される
- 1枚のスライドに1つのメッセージを意識する
- AIが作った内容を自分の言葉に直すことが重要
- デザインは自分で作る必要がある

まずはChatGPTやCopilotに「○○についてプレゼンします。構成を考えてください」と伝えてみてください。
