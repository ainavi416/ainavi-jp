## Midjourneyとは？

Midjourneyは、**最も高品質な画像生成AIの一つ**として知られています。

写真のようなリアルさ、アート風、キャラクター生成など、様々な画風に対応しており、プロのデザイナーやアーティストからの信頼も厚いです。

ただし無料版がなく、**最安で月額$10の課金が必要**です。

## Midjourneyを使うまでのステップ

### ステップ1：Discordアカウントを作成

Midjourneyを使うには、**Discord経由**での利用が必須です。

1. [Discord公式サイト](https://discord.com)にアクセス
2. 「ユーザー登録」をクリック
3. メールアドレスと基本情報を入力
4. 確認メールを確認してアカウント完成

Discordは**ゲーマー向けのチャットアプリ**ですが、Midjourneyも同じプラットフォームを採用しています。

## 画像を生成する方法

### 基本的な使い方

Midjourneyでは、**/imagine**というコマンドを使用します。

**手順：**

1. Discordの**Newbies**チャンネルを開く
2. メッセージ欄に「**/imagine**」と入力
3. スペースを入れて、生成したい画像の説明を入力
4. Enterキーを押す

**例：**
```
/imagine 青い空と白い雲が見える山頂。
朝日が当たっている。デジタルアート風。高品質。
```

30秒〜1分で、**4種類の異なる画像が生成されます**。

![Midjourneyの生成画面](/ainavi-jp/images/midjourney-generation-screen.jpg)

### 画像を精密化する

4つの画像が生成されたら、気に入ったものを選んで精密化できます。

画像の下には、**U（Upscale）ボタン**と**V（Variation）ボタン**があります。

| ボタン | 機能 |
|---|---|
| U1〜U4 | 選んだ画像を大きく、より精密化 |
| V1〜V4 | 選んだ画像に似た別パターンを生成 |

例えば「U1」をクリックすると、1番目の画像がより高品質になります。

## 高品質な画像を作るプロンプトのコツ

Midjourneyは、**細かいプロンプトの指定に応答**するため、コツを知ることで格段に品質が上がります。

### コツ1：アート風を具体的に指定

```
デジタルアート、油絵、アニメ、3DCG、写真、イラスト
```

など、詳細に指定します。

**例：**
```
/imagine 古い図書館の内部。
本棚。窓から光が入っている。
油絵風。ヨーロッパ建築。
```

### コツ2：撮影パラメータを付与

カメラの設定のような指示も効果的です。

```
/imagine 森の中の女性ポートレート。
広角レンズ。自然光。ボケた背景。
映画的。高品質。4K。
```

### コツ3：負のプロンプトを使う（--no）

「このような要素は入れたくない」という指示も可能です。

```
/imagine 美しい風景 --no 人物, 建物, 看板
```
