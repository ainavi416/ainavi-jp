ビジネスメールを書くのに時間がかかる、適切な表現が思いつかない——そんな悩みを解決するのがAIです。

この記事では、ChatGPTやClaudeを使ってビジネスメールを数秒で作成する方法を、実際に使えるプロンプト（AIへの指示文）と例文つきで解説します。

## AIでビジネスメールを作るメリット

AIを使うと以下のメリットがあります。

- **作成時間が10分の1に**: ゼロから考える必要がなくなる
- **適切な表現が自動**: 敬語やビジネス文の型をAIが自動で選ぶ
- **複数パターン作成**: 言い回しを変えた候補を同時に作れる

実際、ビジネスメールの80%は定型業務。AIに任せることで本来の仕事に集中できます。

## AIメール作成に使えるツール

ビジネスメール作成には、以下のAIツールが使えます。

| ツール | 特徴 | 無料範囲 |
|-------|------|---------|
| **ChatGPT** | 汎用性が高く、複数候補作成が得意 | 月40回まで（GPT-4o） |
| **Claude** | 丁寧な文章を作るのが得意 | 無料で使える |
| **Gemini** | Google連携で下書きをGmailに直接送れる | 完全無料 |
| **Copilot** | Windows標準でインストール不要 | 完全無料 |

初めて使うなら、[ChatGPT](https://ainavi416.github.io/ainavi-jp/posts/chatgpt-beginner-guide/)か[Claude](https://ainavi416.github.io/ainavi-jp/posts/claude-beginner-guide/)がおすすめです。どちらも登録後すぐに使えます。

## AIでメールを作る基本手順

どのAIツールでも、以下の3ステップで完成します。

### 1. AIツールを開く

[ChatGPT](https://chat.openai.com/)または[Claude](https://claude.ai/)にアクセスしてログインします。

### 2. プロンプトを入力する

以下の形式でAIに指示を出します。

```
以下の内容でビジネスメールを作成してください。

【相手】〇〇株式会社 △△様
【目的】××の件で依頼
【内容】
- 〜〜について
- 〜〜をお願いしたい
【トーン】丁寧／カジュアル
```

相手・目的・内容・トーンを指定するだけでOKです。

### 3. 内容を確認・修正する

AIが生成したメールを確認し、必要に応じて以下の指示で調整します。

- 「もう少し簡潔にして」
- 「敬語をもっと丁寧にして」
- 「件名も追加して」

これだけで実務レベルのメールが完成します。

## シーン別｜コピペで使えるプロンプト例

実際に使えるプロンプトを5パターン紹介します。

### 【依頼メール】資料送付をお願いする

```
以下の内容で資料送付依頼のメールを作成してください。

【相手】取引先企業の担当者
【目的】製品カタログの送付依頼
【内容】
- 新製品のカタログを送ってほしい
- 来週の提案資料に使用したい
【トーン】丁寧
```

**生成例（ChatGPT）**:

> 件名：製品カタログのご送付をお願いいたします
>
> 〇〇株式会社
> △△様
>
> いつもお世話になっております。
> □□株式会社の■■です。
>
> 来週の提案資料に使用いたしたく、貴社の新製品カタログをご送付いただけますでしょうか。
>
> お忙しいところ恐縮ですが、ご対応のほどよろしくお願いいたします。

### 【アポイント調整】打ち合わせ日程の提案

```
以下の内容で打ち合わせ日程調整のメールを作成してください。

【相手】クライアント企業の担当者
【目的】次回MTGの日程調整
【内容】
- 以下3候補から選んでほしい
  - 2月20日（木）14:00〜
  - 2月21日（金）10:00〜
  - 2月25日（火）15:00〜
- オンラインで実施
【トーン】丁寧
```

### 【お詫び】納期遅延を伝える

```
以下の内容でお詫びメールを作成してください。

【相手】発注元の担当者
【目的】納期遅延のお詫び
【内容】
- 当初予定していた2月15日の納品が2月20日に延期
- 原因は部材の入荷遅れ
- 今後の対策も記載
【トーン】非常に丁寧
```

**生成例（Claude）**:

> 件名：納期遅延のお詫びと新納期のご連絡
>
> 〇〇株式会社
> △△様
>
> いつも大変お世話になっております。
> □□株式会社の■■でございます。
>
> この度、2月15日納品予定の件につきまして、誠に申し訳ございませんが、納期を2月20日に変更させていただきたくご連絡申し上げます。
>
> 原因は部材の入荷遅れによるものでございます。今後このような事態が発生しないよう、仕入先の複数確保と納期管理を徹底してまいります。
>
> ご迷惑をおかけし、深くお詫び申し上げます。
> 何卒ご理解を賜りますよう、よろしくお願いいたします。

### 【問い合わせ】不明点を質問する

```
以下の内容で問い合わせメールを作成してください。

【相手】サービス提供会社のサポート窓口
【目的】契約プランの変更方法を確認
【内容】
- 現在フリープランを使用中
- ビジネスプランへの変更手順を知りたい
- 請求書払いに対応しているか確認したい
【トーン】丁寧
```

### 【お礼】商談後のフォローアップ

```
以下の内容でお礼メールを作成してください。

【相手】本日商談した企業の担当者
【目的】商談のお礼と次回アクション確認
【内容】
- 本日の打ち合わせのお礼
- 提案資料は明日送付予定
- 次回は来週中に再度MTG希望
【トーン】丁寧だが親しみやすく
```

## AIメール作成のコツ3選

より良いメールを作るコツを3つ紹介します。

### コツ1: 具体的な情報を渡す

「メール作って」だけでなく、**5W1H（誰に・何を・いつ・なぜ・どこで・どうやって）を明示**すると精度が上がります。

悪い例：
```
打ち合わせのメールを作って
```

良い例：
```
2月20日14時からのオンラインMTGの日程確認メールを作って。
相手は取引先のAさん。議題はQ1の販促施策について。
```

### コツ2: トーンを指定する

「丁寧」「カジュアル」「簡潔」などの指定で、文体を調整できます。

- 社外・上司向け → **「非常に丁寧」**
- 社内・同僚向け → **「丁寧だがフランク」**
- 急ぎの連絡 → **「簡潔」**

### コツ3: 複数候補を作らせる

「3パターン作って」と指示すると、表現の異なる候補が出ます。比較して最適なものを選べます。

```
以下の内容で、トーンの異なるメールを3パターン作成してください。
（内容省略）
```

## AIでメールを作る際の注意点

便利なAIメール作成ですが、以下の点に注意しましょう。

### 必ず内容を確認する

AIは文法的に正しくても、**事実誤認や不適切な表現**を含む場合があります。送信前に必ず目を通してください。

### 機密情報は入力しない

社外秘の情報・個人情報・契約内容などは、AIに入力しないでください。AIの学習データに使われる可能性があります。

安全な使い方については、[AIに個人情報を入れても大丈夫？安全な使い方](https://ainavi416.github.io/ainavi-jp/posts/ai-safety-privacy/)で詳しく解説しています。

### 定型文化しすぎない

全てのメールをAI任せにすると、相手に機械的な印象を与えます。重要な場面では**自分の言葉を加える**ことをおすすめします。

## まとめ

AIを使えば、ビジネスメールを数秒で作成できます。

**この記事のポイント**:
- ChatGPT・Claudeなら無料で使える

- 送信前の確認と機密情報の非入力は必須
