長いレポートやWeb記事を読む時間がない。そんなときはAIに要約させると便利です。

この記事では、ChatGPTやClaudeを使って文章を要約する方法を、初心者向けに解説します。PDFやWebページもそのまま要約できます。

## AIで要約できる文章の種類

AIが要約できるのは以下のような文章です。

- **Web記事・ニュース**（URLを渡すだけ）
- **PDFファイル**（レポート、論文、契約書など）
- **メールやチャット**（コピペで要約）
- **議事録や会議の書き起こし**

文字数制限はありますが、数千〜数万文字の長文でも数秒で要約してくれます。

![ChatGPTとClaudeを使ってPDFや長文記事を要約している様子](/ainavi-jp/images/ai-summary-concept.jpg)

## ChatGPTで要約する方法

ChatGPTを使った要約の手順は以下の通りです。

### 1. ChatGPTにログインする

[ChatGPT公式サイト](https://chatgpt.com/)にアクセスしてログインします。無料プランでも要約は十分使えます。

### 2. 要約したい文章をコピペする

要約したい文章をコピーして、ChatGPTのチャット欄に貼り付けます。

### 3. 要約の指示を書く

文章の前後に指示文を添えます。

**例文:**

```
以下の文章を300文字程度に要約してください。

（ここに文章をペースト）
```

これだけで要約が返ってきます。

### PDFを要約する場合

ChatGPT Plus（有料プラン）に加入している場合、PDFファイルを直接アップロードできます。

1. チャット欄の左下にある「📎」（クリップマーク）をクリック
2. PDFファイルを選択してアップロード
3. 「このPDFの内容を要約してください」と入力

無料プランの場合は、PDFをテキストに変換してからコピペする必要があります。

ChatGPTの詳しい使い方は[ChatGPTの使い方｜登録から初めての質問まで3分で解説](https://ainavi416.github.io/ainavi-jp/posts/chatgpt-beginner-guide/)をご覧ください。

## Claudeで要約する方法

Claudeは長文の要約が得意です。ChatGPTよりも多くの文字数を一度に処理できます。

### 1. Claudeにログインする

[Claude公式サイト](https://claude.ai/)にアクセスしてログインします。無料プランでOKです。

### 2. 文章をペーストして要約指示を出す

ChatGPTと同じく、文章をペーストして指示を出します。

**例文:**

```
以下の記事を3行で要約してください。

（ここに文章をペースト）
```

Claudeは箇条書きでの要約も得意なので、「箇条書き3つにまとめて」と指示するのもおすすめです。

### PDFやWebページを要約する場合

Claudeは無料プランでもPDFのアップロードに対応しています。

1. チャット欄の「📎」マークをクリック
2. PDFファイルをアップロード
3. 「このPDFの要点をまとめてください」と入力

また、URLを貼り付けるだけでWebページの内容を要約してくれる場合もあります（サイトによっては取得できないこともあります）。

Claudeの詳しい使い方は[Claudeの使い方｜初心者向けに始め方と無料でできることを解説](https://ainavi416.github.io/ainavi-jp/posts/claude-beginner-guide/)をご覧ください。

## 要約の質を上げるコツ

AIにただ「要約して」と伝えるだけでも機能しますが、以下の工夫で精度が上がります。

### 文字数や形式を指定する

「3行で」「300文字で」「箇条書き5つで」のように具体的に指定すると、欲しい形式で出力されます。

**例:**

```
以下の文章を、重要なポイント3つに絞って箇条書きで要約してください。
```

### 目的を伝える

「誰のために要約するか」を明確にすると、AIが必要な情報を選んでくれます。

**例:**

```
以下の契約書の内容を、法律に詳しくない人にもわかるように要約してください。
```

### 要約後に質問する

要約結果を受け取った後、不明点があればそのまま質問できます。

**例:**

```
この要約の「データ保護」の部分について、もう少し詳しく教えてください。
```

AIはチャット形式なので、やり取りしながら理解を深められます。

## ChatGPTとClaudeどちらを使うべき？

要約に使うAIは以下のように選ぶとよいでしょう。

| 用途 | おすすめAI | 理由 |
|------|-----------|------|
| 短い記事（数千文字） | ChatGPT | レスポンスが速い |
| 長文・PDFファイル | Claude | 一度に処理できる文字数が多い |
| ビジネス文書 | Claude | 丁寧で読みやすい文体 |
| ニュース記事 | どちらでもOK | 好みで選んで問題なし |

どちらも無料で使えるので、両方試して使いやすい方を選ぶのもおすすめです。

両者の違いは[ChatGPTとClaudeの違い｜初心者はどっちを使うべき？](https://ainavi416.github.io/ainavi-jp/posts/chatgpt-vs-claude/)で詳しく解説しています。

## まとめ

AIを使えば、長い文章も数秒で要約できます。

**この記事のポイント:**

- ChatGPTやClaudeに文章をコピペして「要約して」と伝えるだけ
- PDFファイルやWebページもそのまま要約できる
- 文字数や形式を指定すると、欲しい形で出力される
- 長文にはClaudeが向いている

まずは気になった記事をコピーして、ChatGPTやClaudeに貼り付けてみてください。
