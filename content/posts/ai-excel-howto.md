Excelの関数がわからない、グラフの作り方がわからない。そんな悩みはAIが解決してくれます。

この記事では、ChatGPTやCopilotを使ってExcel作業を効率化する方法を、初心者向けに解説します。

## AIでできるExcel作業

AIは以下のようなExcel作業を手伝ってくれます。

- **関数の作成**（VLOOKUP、IF、SUMIFSなど）
- **データ分析**（平均、最大値、傾向の読み取り）
- **グラフ作成の手順説明**
- **エラーの原因特定**
- **マクロ・VBAコード生成**

関数名を覚えていなくても、「やりたいこと」を日本語で伝えるだけでOKです。

## ChatGPTでExcel作業を効率化する方法

ChatGPTを使ったExcel効率化の具体例を見ていきましょう。

### 1. 関数を教えてもらう

Excelの関数がわからないときは、「やりたいこと」を伝えるだけで関数を教えてくれます。

**例文:**

```
A列に商品名、B列に売上が入っています。
売上が10万円以上の行だけ「達成」、それ以外は「未達」とC列に表示したいです。
どんな関数を使えばいいですか?
```

**ChatGPTの回答:**

```
C列に以下の関数を入力してください。

=IF(B2>=100000,"達成","未達")

これをC2セルに入力し、下にドラッグすればOKです。
```

このように、関数名を知らなくてもやりたいことを伝えるだけで教えてくれます。

### 2. 複雑な関数を作ってもらう

VLOOKUPやSUMIFSなど、複雑な関数もAIが作ってくれます。

**例文:**

```
A列に社員名、B列に部署、C列に売上があります。
「営業部」の社員の売上合計を計算したいです。
関数を教えてください。
```

**ChatGPTの回答:**

```
以下のSUMIF関数を使ってください。

=SUMIF(B:B,"営業部",C:C)

これで営業部の売上合計が計算できます。
```

条件付き集計や検索など、複雑な処理もAIに任せられます。

### 3. エラーの原因を調べる

Excelで「#VALUE!」や「#REF!」のエラーが出たときも、AIに聞けば原因を教えてくれます。

**例文:**

```
ExcelでVLOOKUP関数を使ったら「#N/A」エラーが出ました。
どうすれば直りますか?

使った関数:
=VLOOKUP(A2,D:F,2,FALSE)
```

**ChatGPTの回答:**

```
#N/Aエラーは「検索値が見つからない」ときに出ます。
以下を確認してください:

1. A2の値がD列に存在するか
2. データ型が一致しているか（数値と文字列が混在していないか）
3. スペースや改行が入っていないか

IFERROR関数で「見つかりません」と表示させることもできます:
=IFERROR(VLOOKUP(A2,D:F,2,FALSE),"見つかりません")
```

エラーメッセージをそのまま貼り付けて質問すれば、原因と解決法を教えてくれます。

ChatGPTの詳しい使い方は[ChatGPTの使い方｜登録から初めての質問まで3分で解説](https://ainavi416.github.io/ainavi-jp/posts/chatgpt-beginner-guide/)をご覧ください。

## Microsoft CopilotでExcelを操作する

Microsoft CopilotはWindows 11に標準搭載されているAIで、Excel作業にも使えます。

### 1. Copilotを起動する

Windows 11の場合、タスクバーのCopilotアイコンをクリックするだけで起動します。

### 2. Excelの質問をする

Copilotに直接Excelの質問ができます。

**例文:**

```
Excelで横軸に月、縦軸に売上の折れ線グラフを作る手順を教えてください。
```

**Copilotの回答:**

```
以下の手順で折れ線グラフを作成できます:

1. グラフにしたいデータ範囲を選択
2. 「挿入」タブ → 「グラフ」 → 「折れ線グラフ」を選択
3. グラフの種類を選んで「OK」

グラフができたら、グラフをクリックして「グラフのデザイン」タブから
デザインを変更できます。
```

Copilotの詳しい使い方は[Microsoft Copilotの使い方｜Windows標準のAIを活用する方法](https://ainavi416.github.io/ainavi-jp/posts/copilot-beginner-guide/)をご覧ください。

### 3. Excel関数をCopilotに聞く

ChatGPTと同じように、Copilotでも関数を教えてもらえます。

**例文:**

```
A列に日付、B列に売上があります。
今月の売上合計を計算したいです。
```

**Copilotの回答:**

```
SUMIFS関数を使います。

=SUMIFS(B:B,A:A,">="&DATE(2026,2,1),A:A,"<"&DATE(2026,3,1))

これで2月の売上合計が計算できます。
```

Windows 11を使っているなら、わざわざブラウザを開かなくてもCopilotでExcelの質問ができます。

## AIにExcel作業を頼むときのコツ

AIにExcel作業を依頼するときは、以下のポイントを意識すると精度が上がります。

### 1. データの構造を伝える

「A列に○○、B列に○○があります」のように、データの配置を伝えると正確な関数を教えてくれます。

**良い例:**

```
A列に商品名、B列に単価、C列に個数があります。
D列に合計金額（単価×個数）を計算する関数を教えてください。
```

**悪い例:**

```
掛け算の関数を教えてください。
```

データの構造を伝えることで、セル参照まで含めた正確な関数が返ってきます。

### 2. スクリーンショットを添付する

ChatGPT PlusやCopilotは画像も読み取れるので、Excelのスクリーンショットを添付すると、より正確な回答が得られます。

1. Excel画面をスクリーンショット（Win + Shift + S）
2. AIのチャット欄に画像を貼り付け
3. 「このExcelファイルで売上合計を計算したいです」と質問

画像を見ながら回答してくれるので、データ構造を言葉で説明する手間が省けます。

### 3. エラーメッセージはそのまま貼る

エラーが出たときは、エラーメッセージや関数をそのままコピペして質問しましょう。

**例:**

```
以下の関数でエラーが出ます。どう直せばいいですか?

=VLOOKUP(A2,Sheet2!A:C,2,0)

エラー: #REF!
```

エラー内容と関数を両方伝えることで、的確な解決法が返ってきます。

<div class="info-box">
<p><strong>ポイント</strong></p>
<p>AIに質問するときは、「やりたいこと」を具体的に伝えるのがコツです。「売上を計算したい」だけでなく、「営業部の今月の売上合計を計算したい」のように詳しく伝えましょう。</p>
</div>

## AIでマクロ・VBAコードを生成する

繰り返し作業を自動化したいときは、AIにVBAコードを書いてもらうこともできます。

### VBAコード生成の例

**例文:**

```
ExcelでA列の空白セルを一括で削除するVBAコードを書いてください。
```

**ChatGPTの回答:**

```vba
Sub DeleteBlankCells()
    Dim lastRow As Long
    Dim i As Long

    lastRow = Cells(Rows.Count, "A").End(xlUp).Row

    For i = lastRow To 1 Step -1
        If Cells(i, "A").Value = "" Then
            Rows(i).Delete
        End If
    Next i
End Sub
```

使い方:
1. Excel で Alt + F11 を押してVBAエディタを開く
2. 「挿入」→「標準モジュール」
3. 上記コードを貼り付け
4. F5で実行

```

プログラミング知識がなくても、やりたいことを伝えるだけでVBAコードを作ってくれます。

<div class="warning-box">
<p><strong>注意</strong></p>
<p>VBAコードは実行前に必ずデータのバックアップを取ってください。誤ったコードを実行するとデータが消える可能性があります。</p>
</div>

## ChatGPTとCopilotどちらを使うべき？

Excel作業に使うAIは以下のように選ぶとよいでしょう。

| 用途 | おすすめAI | 理由 |
|------|-----------|------|
| 関数を教えてもらう | どちらでもOK | 両方とも正確に教えてくれる |
| 複雑な関数（VLOOKUP等） | ChatGPT | 詳しい説明が返ってくる |
| 操作手順を知りたい | Copilot | Excelの画面を見せながら質問できる |
| VBAコード生成 | ChatGPT | より複雑なコードも生成可能 |
| すぐに質問したい | Copilot | Windows標準なのでブラウザ不要 |

どちらも無料で使えるので、両方試して使いやすい方を選ぶのがおすすめです。

## よくある質問

### Q1: AIが作った関数は正しいですか？

A: ほとんどの場合正しいですが、稀に間違っていることもあります。実際にExcelに入力して動作確認してから使いましょう。

### Q2: Excelファイルを直接AIに読み込ませられますか？

A: ChatGPT Plusなら可能です。Excelファイルをアップロードして「このデータを分析してください」と依頼できます。無料版はコピペで対応してください。

### Q3: AIに個人情報を含むExcelデータを見せても大丈夫？

A: 個人情報や機密情報は入力しないでください。AIに質問するときは、ダミーデータに置き換えるか、データ構造だけを伝えるようにしましょう。

AIを使えば、Excel作業がグッと楽になります。

- データ構造を伝えると、より正確な関数が返ってくる
